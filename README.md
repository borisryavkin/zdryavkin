# Dr. Zdryavkin - Персонализированные добавки для здоровья

Веб-сайт для бренда персонализированных пищевых добавок "Dr. Zdryavkin". Сайт разработан с использованием React и Tailwind CSS, с адаптивным дизайном для мобильных и десктопных устройств.

## Особенности проекта

- Адаптивный мобильный дизайн, который легко масштабируется для десктопов
- Современный и чистый интерфейс с использованием зеленых и желтых оттенков
- Полностью настраиваемые компоненты с использованием Tailwind CSS
- Интерактивные элементы с плавными переходами

## Структура проекта

- `Header` - Фиксированный заголовок с логотипом и навигацией
- `Hero` - Главный баннер с заголовком и призывом к действию
- `Benefits` - Раздел с преимуществами (3 колонки)
- `About` - Информация о докторе Здрявкине
- `Testimonials` - Карусель с отзывами клиентов
- `Footer` - Футер с контактной информацией и дисклеймером

## Запуск проекта

```
# Установка зависимостей
npm install

# Запуск проекта в режиме разработки
npm start

# Сборка проекта для продакшена
npm run build
```

## Расширение проекта

Проект структурирован таким образом, что его легко расширить для добавления:
- Страницы блога
- Страницы продуктов
- Формы подписки
- И другие функциональные возможности
